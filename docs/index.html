What is Flint?

Flint was originally written as a simple collection of compilers for large scale client-side javascript applications.

Flint is now also:

  1. A command line tool which can be used to build, compile, and serve your applications.
  2. A core client side and server side library designed to help you stay DRY.
  3. A NodeJS server with Socket.io built-in for easy realtime data exchange capability

Feel free to dive right in or for more information and all the source code visit the Github Repo page.
