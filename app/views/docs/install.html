How to install Flint:

Get Node and npm

Before you can install Flint you need download and install NodeJS. The fastest way to install npm is like so:

  curl https://npmjs.org/install.sh | sh

Install Flint via npm (recommended):

The most stable version of Flint will always live on npm. Install Flint like this: 

   npm install flint -g

Clone the git repo:

You can always clone the latest from github (especially if you'd like to contribute). 

  git clone https://github.com/cif/flint.git
  cd flint
  npm install . -g

NOTE: You will need to install flint with -g if you want the command line tools!
