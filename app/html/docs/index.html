What is Flint?

Flint was originally a collection of compilers (Brunch.io etc.) designed to organize complex client-side Javascript applications written in Coffeescript, Handlebars and Stylus.

Flint is now a npm module that comes with:

  1. A command line tool which can be used to build, compile, and serve your applications.
  2. An "out of the box" REST API requiring ZERO code to implement.
  3. A core class library designed to help you stay DRY on both server and client.
  4. Socket.io built-in for easy realtime data exchange capability.
  5. A simple router and server side templates to serve pages in a traditional style for SEO.

Feel free to dive right in or for the source code check out the Github repo. Don't like reading? Peep the videos.
