How to install Flint:

1. Get NodeJS and npm

Before you can install Flint you'll need to download and install NodeJS.   Next, if you haven't already, install npm with this handy one-liner:

curl https://npmjs.org/install.sh | sh

NOTE: This will only work for *nix operating systems. Further, the uglify-js dependency requires *nix. To request a windows workaround, post an issue to github.

2. Install Flint via npm (recommended):

The most stable version of Flint will always live on npm. Install Flint like this: 

npm install flint -g

Clone the git repo (optional):

You can clone the latest from github (recommended if you'd like to contribute). 

git clone https://github.com/cif/flint.git
cd flint
npm install . -g

NOTE: You will need to install flint with -g if you want the command line tools.
